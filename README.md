# POS Cinema - Refactorización

Este es un proyecto de refactorización de un sistema de Punto de Venta (POS) para cines, migrando de una arquitectura monolítica existente hacia una más limpia, modular y mantenible.

## Descripción

El objetivo principal de este proyecto es modernizar la base del código, mejorar la experiencia de usuario (UX) y la interfaz de usuario (UI), y establecer una arquitectura robusta que facilite el mantenimiento y la adición de nuevas funcionalidades en el futuro.
